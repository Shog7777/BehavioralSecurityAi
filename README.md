# BehavioralSecurityAi
هدف هذا المشروع إلى تطوير نموذج أولي لنظام يعتمد على تقنيات الذكاء الاصطناعي والرؤية الحاسوبية لتحليل لغة الجسد وتعابير الوجه لدى المسافرين في البيئات الأمنية مثل المطارات. يقوم النظام بتحليل الفيديو الملتقط عبر الكاميرات لاكتشاف مؤشرات سلوكية قد تدل على توتر غير طبيعي أو سلوك مريب
# نظام تحليل التوتر في المطارات باستخدام الذكاء الاصطناعي

## 📋 نظرة عامة
نظام ذكي لتحليل لغة الجسد وتعابير الوجه للمسافرين في المطارات، بهدف مساعدة أفراد الأمن في اكتشاف حالات التوتر والقلق غير الطبيعية التي قد تشير إلى نوايا سيئة مثل التهريب أو الاحتيال. يعتمد النظام على تقنيات الرؤية الحاسوبية وتعلم الآلة لتحليل تعابير الوجه وحساب نسبة التوتر بشكل فوري من خلال الكاميرا أو ملفات الفيديو المرفوعة.

## 💡 فكرة المشروع
المشروع مستوحى من أنظمة حقيقية تستخدم في مطارات دول مثل الولايات المتحدة والبرازيل والمكسيك، حيث يعتمد رجال الأمن على قراءة لغة الجسد لتحديد المشتبه بهم قبل تفتيشهم. يقوم نظامنا بأتمتة هذه العملية باستخدام الذكاء الاصطناعي، مما يوفر أداة مساعدة للضباط لزيادة دقة التقييم وتقليل الأخطاء البشرية.

## 🚀 الميزات الرئيسية
- تحليل تعابير الوجه: استخراج مؤشرات التوتر من ملامح الوجه.
- حساب نسبة التوتر: مع تدريج لوني (أخضر منخفض، أصفر متوسط، أحمر مرتفع).
- كاميرا مباشرة: تحليل فوري للمسافرين أمام الكاميرا مع عرض النتائج على الشاشة.
- رفع ملفات فيديو: تحليل مقاطع فيديو مسجلة وعرض إحصائيات مفصلة (متوسط التوتر، أعلى توتر).
- واجهة مستخدم عربية احترافية: تصميم جذاب وسهل الاستخدام يعمل على جميع المتصفحات.
- خصوصية كاملة: جميع العمليات تتم محلياً على جهاز المستخدم، ولا يتم إرسال أي بيانات خارجياً.

## 🛠️ التقنيات المستخدمة
- اللغة: Python 3.10
- إطار العمل: Flask (لبناء تطبيق الويب)
- الرؤية الحاسوبية: OpenCV (كشف الوجوه ومعالجة الصور)
- تحليل المشاعر: DeepFace (نموذج متقدم لتحليل تعابير الوجه)
- الحوسبة الرقمية: NumPy
- الواجهة الأمامية: HTML5, CSS3, JavaScript (مع Font Awesome للأيقونات)
